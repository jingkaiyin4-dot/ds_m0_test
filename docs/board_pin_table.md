# 板子引脚表与画板注意事项

## 1. 说明

这份文档用于你后续画板，内容包含：

- 当前工程已经实际使用的引脚
- 左右减速编码电机引脚
- IMU / TOF / Camera / 屏幕引脚
- 按键引脚
- 推荐预留与画板注意事项

本文档以当前代码和 `syscfg` 配置为准。

## 4. 按键建议

### 当前已经有的按键

| 按键 | 引脚 | 用途 |
|---|---|---|
| `KEY0` | `PB21` | 板载按键，当前用于模式切换 |

### 推荐新增按键

| 按键 | 引脚 | 建议用途 |
|---|---|---|
| `KEY1` | `PB16` | 调试/模式切换 |
| `KEY2` | `PB17` | 调试/参数切换 |

### 按键接法建议

推荐统一按键接法：

- 按键一端接 GPIO
- 另一端接 `GND`
- 软件开上拉
- 软件做消抖

### 电容建议

你的使用场景主要是：

- 调试
- 调参
- 模式切换

不是比赛中高频按键操作，所以：

- 可以不加电容
- 如果想更稳，可加 `0.1uF`
- `1uF` 也能用，但响应会更钝一点
- 不建议按键上用 `10uF`

## 5. 画板注意事项

### 5.1 左右电机和编码器丝印一定写清楚

建议丝印直接写：

右轮：

- `R_PWM PB4`
- `R_IN1 PA30`
- `R_IN2 PA29`
- `R_ENC_A PB3`
- `R_ENC_B PB5`

左轮：

- `L_PWM PB1`
- `L_IN1 PB0`
- `L_IN2 PB2`
- `L_ENC_A PB6`
- `L_ENC_B PB7`

### 5.2 编码器和电机接口最好分开

不要把：

- 电机两根动力线
- 编码器 4 根信号/供电线

混在一个不清晰的接口定义里。

建议：

- 电机接口单独 2Pin
- 编码器接口单独 4Pin 或 5Pin

### 5.3 编码器 A 相最好靠近中断脚定义不变

当前软件是这样用的：

- 右轮 A 相：`PB3`
- 左轮 A 相：`PB6`

这是为了用 GPIO 中断统计脉冲。

画板时尽量不要交换这两个 A 相和 B 相的角色，不然软件要再改。

### 5.4 所有地必须共地

必须保证以下全部共地：

- 主控 `GND`
- 电机驱动模块 `GND`
- 编码器 `GND`
- 步进电机驱动 `GND`
- 传感器 `GND`
- 电池负极

否则会出现：

- 电机偶发抽动
- 编码器计数乱跳
- 串口通信不稳
- I2C 异常

### 5.5 电机电源和逻辑电源要分清

减速电机和步进电机都会带来较大干扰。

建议：

- 电机电源走单独粗线
- 主控和传感器的供电尽量干净
- 电机驱动附近加足够去耦

### 5.6 屏幕背光脚 `PB26`

如果你后面不需要动态调背光亮度，`BLK` 可以直接固定常亮。

但既然当前已经给它分了脚位，建议板子上还是先保留引出，避免后面想控背光又没法改。

### 5.7 调试测试点强烈建议预留

建议至少给这些脚留测试点：

- `PB4` 右轮 PWM
- `PB1` 左轮 PWM
- `PA30/PA29` 右轮方向
- `PB0/PB2` 左轮方向
- `PB3/PB5` 右轮编码器 A/B
- `PB6/PB7` 左轮编码器 A/B
- `PA28/PA31` I2C
- `PA8/PA9` Camera/TOF UART

### 5.8 模式选择脚 `PA12`

这个脚当前可以二选一：

1. 做 `CAM/TOF` 硬件拨位输入
2. 改成新增按键 `KEY1`

画板前最好先定下来，不然功能会冲突。

### 5.9 `PB16` 和 `PB17` 适合作为新增调试按键

当前配置下，`PB16` 和 `PB17` 没有和核心外设冲突，适合做：

- 新增调试按键
- 模式切换按键

注意不要和 `PA16` 混淆。当前工程里占用的是 `PA16`，不是 `PB16`。

## 6. 当前最推荐的按键方案

如果你要保留板载按键并再加两个按键，建议：

- `KEY0 -> PB21` 已有板载按键
- `KEY1 -> PB16`
- `KEY2 -> PB17`

这样软件结构也最好统一。

## 7. 建议你定版前再次确认的事项

1. `PA12` 最终是否保留做模式拨位
2. `PB26` 背光脚是否要保留软件控制
3. 左右电机接口丝印是否和代码定义一致
4. 编码器 A/B 接口方向标识是否清晰
5. 是否预留关键测试点

## 8. 简要结论

当前板子画图时，最重要的底盘与按键引脚建议是：

- 右轮：`PB4 / PA30 / PA29 / PB3 / PB5`
- 左轮：`PB1 / PB0 / PB2 / PB6 / PB7`
- 按键：`PB21`、`PB16`、`PB17`

这套分配和当前代码是兼容的，后续主要需要调的是方向符号和 PID，不是引脚映射本身。
